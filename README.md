# Example API Test Project

## General info
This project is basic api test project.

##Prerequisites
1. JDK 1.8 or above
2. Maven 3 or above
3. Authentication is not required for the api

###Language
1. JAVA

### Used Tools and Frameworks
1. Rest-Assured
2. Maven repo
3. TestNG
4. POM

### API Project URL
```
https://petstore.swagger.io/
```

##Run Tests
```
mvn clean test
```
